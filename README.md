# MiniGit

A lightweight Git clone built in C++ to understand how Git works internally.

## 🧱 Features (Planned)
- `init` – create a new repository
- `add` – stage files
- `commit` – save snapshots
- `log` – view commit history

## ⚙️ Build
```bash
g++ mini.cpp commands/*.cpp utils/*.cpp -o minigit
